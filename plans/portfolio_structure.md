# Структура веб-сайта портфолио для IT-специалиста

## Общие требования
- Одностраничный лендинг
- Минималистичный дизайн
- Адаптивный интерфейс под все устройства
- Современные UI/UX решения
- Плавная навигация
- Профессиональная цветовая палитра (синий, серый, черный, акценты зеленого/оранжевого)
- Четкая типографика

## Структура сайта

### 1. Шапка (Header)
- Логотип/имя специалиста
- Навигационное меню (плавная прокрутка к секциям)
- Кнопка "Связаться со мной"

### 2. Заголовок (Hero Section)
- Приветствие: "Привет, я Алексей Петров"
- Должность: "Full-Stack разработчик"
- Краткое описание: "Создаю современные веб-приложения с использованием передовых технологий"
- Призыв к действию: "Посмотреть мои проекты"
- Социальные ссылки

### 3. Обо мне (About)
- Фотография профиля
- Подробное описание опыта и подхода
- Ключевые компетенции

### 4. Навыки (Skills)
- Технологии: JavaScript, React, Node.js, Python, SQL
- Уровень владения каждым навыком (в виде прогресс-баров)
- Дополнительные навыки: Agile, Git, CI/CD и т.д.

### 5. Проекты (Projects)
- Карточки проектов с изображениями
- Названия, описания и технологии
- Ссылки на GitHub и живые демо

### 6. Контакты (Contact)
- Форма обратной связи
- Контактная информация
- Социальные сети

### 7. Футер (Footer)
- Копирайт
- Ссылки на социальные сети
- Политика конфиденциальности

## Технические требования
- HTML5 семантическая разметка
- CSS3 Flexbox/Grid
- Анимации и переходы
- JavaScript для интерактивности
- Адаптивность: мобильные, планшеты, десктопы
- SEO-оптимизация
- Быстрая загрузка страницы

## Цветовая палитра
- Основной: #0a192f (темно-синий)
- Акцент: #64ffda (бирюзовый)
- Текст: #ccd6f6 (светло-серый)
- Вторичный текст: #8892b0 (нейтральный серый)
- Фон: #112240 (темно-синий)

## Шрифты
- Основной: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif
- Моноширинный: 'Courier New', Courier, monospace (для кода)
